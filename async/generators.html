Generators¶

Write me.

kaa.generator(generic=False)¶

This decorator is used to construct asynchronous generators, to be used in combination with functions that return InProgress objects (such as those functions decorated with kaa.coroutine() or kaa.threaded()).

Parameter:generic – if True, a Generator object is passed as the first argument to the decorated function. This can be used for functions which have not been previously decoratored with a supported decorator (such as @coroutine or @threaded).
